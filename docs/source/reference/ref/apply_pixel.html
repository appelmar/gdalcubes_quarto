# apply_pixel

Apply a function over (multi-band) pixels



```{r include=FALSE}
library(gdalcubes)
```

## Description

This generic function applies a function on pixels of a data cube, an R array, or other classes if implemented.


## Usage


```r
apply_pixel(x, ...)

```
